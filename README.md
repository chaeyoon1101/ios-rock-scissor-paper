## 코드 구현
- 메뉴를 출력하는 `printGameMenu` 함수 구현
- 컴퓨터의 임의의 수를 만드는 `makeComputerValue` 함수 구현
- 사용자의 수를 입력받는 `inputUserValue` 함수 구현
- 사용자의 수가 입력규칙과 일치하는지 확인하는 `checkUserValue` 함수 구현
- 승리와 패배를 확인하는 `checkWinOrLose` 함수 구현
